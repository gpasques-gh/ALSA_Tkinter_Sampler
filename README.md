# ALSA Sampler for WAVE files with Python Tkinter GUI

## Dependencies
- alsasoundlib
- rtmidi
- tkinter

## Building instructions
- make 
- make INPUT=KB_INPUT (for keyboard input)
- make CONTROLLER=AKAI_MPK (for using an AKAI MPK Mini, testing purposes)

## Use instructions
On Linux systems, you can just build the project then launch the file python/main.py
The project is really starting, so a lot of basic GUI features are missing and some parameters are heavily hard coded. Use at your own will.
